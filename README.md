Application pour gérer les loadouts de Black Ops 7.

## 🎯 Fonctionnalités

- ✅ 43 armes disponibles
- ✅ Interface CustomTkinter moderne
- ✅ Sauvegarde des loadouts
- ✅ Gestion des configurations

## 📦 Installation

\\\ash
# Cloner le repo
git clone https://github.com/DocShakaGame/bo7-loadout-manager.git
cd bo7-loadout-manager

# Créer un venv (optionnel)
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# Installer les dépendances
pip install -r requirements.txt

# Lancer l'app
python main.py
\\\
